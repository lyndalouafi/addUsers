# Users API (Backend)

## 🚀 Description
RESTful API built with Express.js for managing users.

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- CORS

## 📌 Features
- Get all users
- Search users by firstname
- Create user
- Update user
- Delete user

## 🔗 Endpoints
- GET /users
- GET /users/search?q=...
- POST /users
- PUT /users/:id
- DELETE /users/:id
